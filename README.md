# coopdevs-landing-page

- Use node 6 or higher (`nvm use` if you use `nvm`).

## Development

- `npm install`
- `npm start`
- Go to `http://localhost:3000`

## Deployment

- `npm install`
- `npm run build`
- `npm run export`
- Commit and push your changes :D
- `npm run publish` (will push `out` into `gh-pages` subtree).
